# AgentFlow

**Visual AI Workflow Builder** - Build AI automations visually. No code required.

[![CI](https://github.com/mpalmer79/agentflow/actions/workflows/ci.yml/badge.svg)](https://github.com/mpalmer79/agentflow/actions/workflows/ci.yml)

## Overview

AgentFlow is a drag-and-drop workflow builder that lets you create AI-powered automations by connecting visual nodes. From personal task automation to enterprise-scale operations.
